What

This is a very simple ChordPro-to-text converter. All it does, is extract the inline chord symbols from the text and place it above its original position.

Example:

This is a [em]stupid demo [am] example
with two [am7]stupid demo [C] lines.

becomes:

          [em]       [am]
This is a stupid demo example

         [am7]      [C]
with two stupid demo lines.

Installation

{% highlight bash %} gem sources -a http://Gems.GitHub.Com/ gem install iconofthestoneage-chord {% endhighlight %}

Usage

{% highlight bash %} chord inputfile.txt -o outputfile.txt {% endhighlight %}
